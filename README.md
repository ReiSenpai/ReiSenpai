## 🧠 Sobre mí: Ingeniería, Diseño y Automatización

¡Hola! Soy **Rei Senpai**, un **Ingeniero de Software FullStack** con la visión de un **UX/UI Designer** y la eficiencia de un profesional de **DevOps**. Mi pasión es construir sistemas completos: desde el diseño visual impecable hasta la automatización de su despliegue en producción.

- 💼 **Especialidades y Enfoque Principal**:
  - **Desarrollo Frontend & UX/UI**: Diseño de interfaces limpias y accesibles, prototipado rápido y microinteracciones de alto impacto.
  - **Ingeniería de Software FullStack**: Creación de soluciones robustas y escalables con un enfoque en la calidad del código.
  - **Infraestructura & Automatización (DevOps)**: Gestión de CI/CD, contenedorización y despliegues eficientes en la nube.

---

## 🛠️ Stack de Herramientas Completo

Mi arsenal de herramientas abarca todo el ciclo de vida del software, garantizando calidad desde el wireframe hasta el despliegue.

- 🎨 **Diseño UX/UI**: Figma, Balsamiq, Adobe XD
- 💻 **Frontend**: HTML5, CSS3, JavaScript/TypeScript, **Angular**, **React**, **Next.js**
- 🔧 **Backend**: Java (**Spring Boot**), Python (**Flask/Django**), Node.js (Express), Spring Security
- 🗃️ **Bases de Datos**: MySQL, PostgreSQL, SQLite

### ☁️ Infraestructura & DevOps

Como entusiasta de DevOps, utilizo las siguientes herramientas para asegurar un flujo de trabajo ágil y robusto:

- **Contenedores y Orquestación**: Docker, Kubernetes
- **Integración y Despliegue Continuo (CI/CD)**: Jenkins, GitHub Actions
- **Cloud & IaaS**: AWS (S3, EC2), Google Cloud, Terraform
- **Control de Versiones**: Git, GitHub

## 🎨 Sección de Diseño y Experiencia de Usuario

Mi motor es la creación de **experiencias digitales que se sienten vivas, no solo funcionales**. Actualmente, estoy enfocado en la fusión de diseño avanzado y accesibilidad:

🧪 **Proyectos de Diseño y Animación Actuales**:

- **🧠 Red de interfaces accesibles UX** para personas con movilidad reducida  
- **📱 SmartVoice V2** – app de control por voz con estilo móvil  
- **🌐 Portfolio animado** con scroll dinámico y microinteracciones  
- **🧩 UI Kits personalizados** en Figma (*light/dark neumorphism*)

🎞️ **Tendencias que exploro constantemente**:

- ✨ *Typing animations* con scroll reveal (**GSAP**, **AOS**)
- 🎯 Interfaces con *microinteracciones reactivas* (hover, click, drag)
- 💡 Diseño visual basado en **glassmorfismo**, **neumorfismo** y **accesibilidad**
- 📽️ Animaciones SVG y loaders personalizados para apps
